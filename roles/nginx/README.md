# nginx

An Ansible role that installs and configures [nginx](https://www.nginx.com/).
By default:
 - Runs on port `80`
 - Copies O2 Web UI catalogue page

## Host Variables (optional)

| Variable                     | Default value       | Notes                                         |
| ---------------------------- | --------------------|---------------------------------------------- |
| nginx_port                   | `80`                | -                                             |
